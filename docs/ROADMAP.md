# Nexus Core Roadmap

## Current Phase

Phase 2F focuses on product completion, folder import foundation, admin maturity, onboarding, settings, multilingual polish, and SaaS readiness.

## Next Recommended Phase

Phase 2G should add organization/workspace permissions and secure admin RPCs before any execution system is introduced.

## Future Phases

- GitHub repository connection without execution
- Persistent project indexing and versioned manifests
- Enterprise team roles and workspace permissions
- Usage snapshots and billing-provider preparation
- Approval workflow for risky operations
- Execution sandbox design review
- Verification engine design
- Audit log export and compliance reporting

## Still Unsupported

- Shell access
- Docker or sandbox workers
- Dependency installation
- Autonomous code modification
- Pull request generation
- Stripe checkout
- Embeddings/vector search
